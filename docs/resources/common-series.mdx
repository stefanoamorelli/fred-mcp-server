# Common FRED Series Reference

Quick reference guide for the most frequently used FRED economic data series.

## Major Economic Indicators

### GDP & Growth

| Series ID | Description | Frequency | Units |
|-----------|-------------|-----------|-------|
| **GDP** | Gross Domestic Product | Quarterly | Billions of Dollars |
| **GDPC1** | Real GDP | Quarterly | Billions of Chained Dollars |
| **GDPPOT** | Real Potential GDP | Quarterly | Billions of Chained Dollars |
| **NYGDPMKTPCDWLD** | World GDP Per Capita | Annual | Constant 2015 US$ |
| **GDPDEF** | GDP Deflator | Quarterly | Index 2017=100 |

## Usage Tips

### Finding Similar Series

1. Use the base series ID to search for related data
2. Add suffixes for different frequencies or adjustments
3. Check tags for geographic or topical variations

### Common Transformations

```json
// Year-over-year inflation
{
  "series_id": "CPIAUCSL",
  "units": "pc1"
}

// Quarterly GDP growth
{
  "series_id": "GDPC1",
  "units": "pca"
}

// Monthly change in employment
{
  "series_id": "PAYEMS",
  "units": "chg"
}
```

### Combining Series

Many analyses require multiple series:

- **Taylor Rule**: DFF, GDPPOT, GDPC1, PCEPI
- **Yield Curve**: DGS3MO, DGS2, DGS10, DGS30
- **Labor Market Health**: UNRATE, PAYEMS, CIVPART, CES0500000003
