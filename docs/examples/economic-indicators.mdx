# Economic Indicators Examples

Learn how to retrieve and analyze key economic indicators using the FRED MCP Server.

## Labor Market Indicators

### Unemployment Rate

Retrieve the monthly unemployment rate:

```javascript
// Using fred_get_series
{
  "series_id": "UNRATE",
  "observation_start": "2020-01-01",
  "observation_end": "2024-10-01"
}
```

**Response:**
```json
{
  "series_id": "UNRATE",
  "title": "Unemployment Rate",
  "units": "Percent",
  "frequency": "Monthly",
  "seasonal_adjustment": "Seasonally Adjusted",
  "observations": [
    {
      "date": "2024-09-01",
      "value": "4.2"
    },
    {
      "date": "2024-08-01",
      "value": "4.3"
    }
  ]
}
```

### Labor Force Participation

Track workforce participation trends:

```javascript
{
  "series_id": "CIVPART",
  "observation_start": "2015-01-01"
}
```

<Tip>
  Combine multiple series to understand labor market dynamics:
  - **UNRATE**: Unemployment rate
  - **CIVPART**: Labor force participation
  - **EMRATIO**: Employment-population ratio
  - **U6RATE**: Broader unemployment measure
</Tip>

## Searching for Indicators

Find indicators by topic:

```javascript
// Search for inflation indicators
{
  "query": "inflation",
  "tags": ["monthly", "sa"],
  "limit": 20,
  "sort_by": "popularity"
}

// Search for employment
{
  "query": "employment",
  "tags": ["bls", "monthly"],
  "limit": 20
}
```

## Best Practices

<Warning>
  **Key considerations when working with indicators:**

  - **Frequency**: Match series frequencies for comparisons
  - **Seasonal Adjustment**: Use SA versions for trend analysis
  - **Units**: Verify units before calculations
  - **Revisions**: Economic data undergoes revisions
  - **Release Dates**: Check when data is typically updated
</Warning>

### Combining Multiple Indicators

```javascript
// Create dashboard of key indicators
const dashboard = [
  { id: "UNRATE", name: "Unemployment" },
  { id: "CPIAUCSL", name: "CPI", transform: "pc1" },
  { id: "GDP", name: "GDP", transform: "pca" },
  { id: "DFF", name: "Fed Funds Rate" },
  { id: "SP500", name: "S&P 500" }
];

// Fetch all in date range
dashboard.forEach(indicator => {
  fred_get_series({
    series_id: indicator.id,
    observation_start: "2020-01-01",
    units: indicator.transform || "lin"
  });
});
```
