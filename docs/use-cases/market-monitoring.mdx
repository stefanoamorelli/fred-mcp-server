# Market Monitoring

Build monitoring systems and dashboards using FRED for real-time economic and market surveillance.

## Dashboard Components

### Key Indicators

```javascript
const dashboard = {
  // Markets
  sp500: "SP500",
  vix: "VIXCLS",
  ten_year: "DGS10",

  // Economy  
  unemployment: "UNRATE",
  inflation: "CPIAUCSL",
  gdp: "GDP",

  // Policy
  fed_funds: "DFF",
  balance_sheet: "WALCL"
};
```

## Alert Systems

Monitor for significant changes:

- Unemployment rate changes > 0.3%
- Inflation exceeds target ranges
- Yield curve inversions
- Market volatility spikes

## Update Frequencies

| Series | Update Frequency | Release Time |
|--------|-----------------|--------------|
| Fed Funds | Daily | Next day |
| Treasury Yields | Daily | Next day |
| Initial Claims | Weekly | Thursday 8:30 AM ET |
| CPI | Monthly | ~13th, 8:30 AM ET |
| Employment | Monthly | First Friday, 8:30 AM ET |
| GDP | Quarterly | ~30 days after quarter |
