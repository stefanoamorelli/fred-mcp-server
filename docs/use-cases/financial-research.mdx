# Financial Research Use Case

Use FRED MCP Server for financial markets analysis, portfolio research, and investment strategy development.

## Market Analysis

### Equity Markets

```javascript
// Major indices
const indices = {
  sp500: "SP500",
  dow: "DJIA",
  nasdaq: "NASDAQCOM",
  russell2000: "RU2000PR",
  wilshire5000: "WILL5000PRFC"
};
```

```javascript

const treasuries = {
  three_month: "DGS3MO",
  two_year: "DGS2",
```
